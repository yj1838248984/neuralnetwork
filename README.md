# neuralnetwork
一个简单的神经网络,采用javascript开发
你可以在浏览器端或者node.js环境中运行

该神经网络采用梯度下降算法和反向传播算法

参数更新采用minibatch

没有太多样例,也没在效率上过多考虑,毕竟是个玩具神经网络

参考资料：
http://neuralnetworksanddeeplearning.com
